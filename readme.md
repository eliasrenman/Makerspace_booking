
NTI Umeås Bokningssytem för makerspace
==

Bokningssystemet som används internt för NTI-gymnasiet i umeå för makerspace.

Teknologier
--
Projektet använder sig av dessa teknologier:
* Laravel
* PHP
* SQL (MySQL)
* javascript
* Jquery
